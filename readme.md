# 🛒 E-Commerce Website (Frontend)

## 🚀 Project Overview

This project is an **industry-oriented E-Commerce frontend website** developed as part of the **RISE Internship – Web Development Program**.

The application focuses on delivering a **clean, responsive, and user-friendly shopping experience**, addressing common issues like poor navigation, slow interfaces, and confusing layouts.

---

## 🎯 Objective

To design and develop a **responsive e-commerce frontend** that:

* Enhances user experience
* Simplifies product discovery
* Implements interactive shopping features

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3 / Tailwind CSS** – Styling and responsive UI
* **JavaScript (ES6)** – Dynamic functionality and cart logic
* **Git & GitHub** – Version control

---

## ✨ Features

* 🛍️ Product listing page
* 🔍 Search functionality
* 🗂️ Category-based filtering
* 🛒 Shopping cart system (JavaScript-based)
* ➕ Add to cart functionality
* 🔢 Quantity management
* 💰 Total price calculation
* 📱 Fully responsive design
* 🎨 Interactive UI with hover effects
